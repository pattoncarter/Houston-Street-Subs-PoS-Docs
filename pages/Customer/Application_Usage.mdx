# Navigation
Item categories can be navigated using the left side navigation bar. This navigation bar features items by category and allows the user to see specific groups of items for convenient access. 
<img src={"@images/customer.png"} alt="Customer View" />
Find your favorite item and click on it to go to the ordering screen! If you have added your entire order you can navigate to the checkout screen and cart view by clicking the shopping cart in the top left corner!
