# Purpose
This API is used as a secure SQL query to checkout the passed items from the Houston Street Subs Database. This will reflect across a few different tables so that manager databases are updated as well as the customer view.

# Parameters
This API requires a slug parameter to call. It can be reached using /api/Checkout/[...slug]. This slug is to be one or more item ids of a product in the table.

I.e. /api/Checkout/221/222/223 will checkout the items with ID 221, 222, and 223.
